# Tạo Tunnel Token mới từ Dashboard

## Cách tạo:

1. Vào **Cloudflare Dashboard**
2. Chọn **Zero Trust** → **Access** → **Tunnels**
3. Chọn tunnel của bạn
4. Click **Configure** (biểu tượng bánh răng)
5. Click tab **Token**
6. Click **Generate token**
7. Copy token mới và gửi cho tôi

---

## Lưu ý:

- Token mới sẽ có ingress rules từ Dashboard
- Token cũ từ máy local không có ingress rules đó
